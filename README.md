# Claude Code Plugin

My personal Claude Code plugin for learning plugin development and automating common workflows.

## Commands

- `/review-code` - Run automated code reviews on uncommitted changes, commits, or GitHub PRs
- `/create-pr` - Create pull requests with AI-generated descriptions

## Structure

```
.claude-plugin/plugin.json   # Plugin metadata
commands/                    # Slash commands
skills/                      # Skill implementations
```
